Here is what i have learned so far.

  1. Make sure your computer has 16 gig memory
  2. Set up Visual Code Studio from the web. Download system installer 64 bit.
  3. Download git scm and install on the PC (64 bit). Use viasula studio as default editor. Use git from windows command prompt. Use windows consol
  4. Go to git.com and set up an account using the same email address.
  5. Check on your PC if git is installed properly. Open git terminal and do set email ==git hub
  6. Then do git config --global user.email your emailID
  7. Verify git config --list. This should show up as user.email = my emailID
